# THE6ISLAND.BLOG - 24/7 News Bot

This repository contains an automated news bot that runs on GitHub Actions.

## Features
- Runs every 4 hours automatically
- Fetches news from NewsData.io
- Posts to WordPress automatically
- Stores logs in Supabase
- Runs even when your devices are off

## Setup
1. Add required secrets in GitHub repository settings
2. The bot runs automatically on schedule
3. Can be manually triggered from GitHub Actions tab

## Secrets Required
- `SUPABASE_URL`
- `SUPABASE_KEY` 
- `NEWSDATA_KEY`
- `WP_TOKEN`
- `SITE_ID`
- `GROQ_KEY`

## Manual Run
Go to Actions → 24/7 News Bot → Run workflow
